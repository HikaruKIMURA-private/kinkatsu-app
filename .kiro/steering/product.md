# Product Overview

筋トレ記録アプリ「Gym Training Log」は、ユーザーが日々のワークアウトを記録・管理し、トレーニングの進捗を追跡できるWebアプリケーションです。

## Core Capabilities

1. **ワークアウト記録**: 日付ごとにワークアウトを記録し、種目、セット、重量、回数、RPE（主観的運動強度）を管理
2. **種目管理**: 部位別（胸、背中、脚、腹筋、腕、肩、前腕、ふくらはぎ）に種目を分類・管理
3. **進捗追跡**: 過去のワークアウトデータを参照し、トレーニングの進捗を可視化
4. **認証・セキュリティ**: Better Authによる安全な認証（メール/パスワード、GitHub OAuth、Passkey対応）

## Target Use Cases

- **日常的なトレーニング記録**: ジムでのワークアウトをその場で記録
- **進捗の確認**: 過去の記録を参照し、重量や回数の推移を確認
- **トレーニング計画**: 日々のワークアウトデータを基に、今後のトレーニング計画を立てる

## Value Proposition

- **シンプルな記録**: 直感的なUIで、トレーニング中でも素早く記録可能
- **データの永続化**: クラウドベースでデータを安全に保存
- **柔軟な種目管理**: ユーザーが独自の種目を追加可能
- **多様な認証方法**: メール/パスワード、GitHub、Passkeyから選択可能
